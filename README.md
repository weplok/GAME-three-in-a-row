# GAME-three-in-a-row

### Pygame Project by students of Yandex Lyceum 2024

## To run game in Windows, use terminal in project-directory:

### Create and run venv

```bash
python -m venv venv
```

```bash
call venv/Scripts/activate
```

### Install package 

```bash
pip install -r requirements/prod.txt
```

### Run game

```bash
cd game
```

```bash
python main.py
```
